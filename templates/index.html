About Us

this web-app helps you quickly analyze a skin lesion. It works in a similar way to a textbook, where you might find an image that looks like your skin problem. But while looking this up in a textbook would take a lot of time, this web-app does it in an instant!
this web-app is designed to be more than a one-disease app. We have trained 10,015 diagnosed images to detect 7 skin lesions.
Our aim is to provide a simple, inexpensive software system to identify potentially all types of skin lesions using a web-app-based platform.
The system applies deep learning–based image analysis to automatically identify and classify different skin lesions from uploaded images.

How it Works

The Algorithm

The algorithm in this project works in a manner similar to an image-search system but is specifically designed for analyzing skin images. Instead of scanning the internet, it uses a curated dataset of clinically diagnosed skin lesion images. When a user uploads an image, it is preprocessed and analyzed using a Convolutional Neural Network (CNN). The CNN extracts key visual features such as color, texture, and shape of the lesion. These features are compared with learned patterns from the training dataset. Based on this comparison, the system predicts the most likely skin disease. The final output includes the predicted class along with confidence level and relevant guidance.

The Database

The system uses the HAM10000 dataset, which contains over 10,000 clinically diagnosed dermoscopic images of both benign and malignant skin lesions, including melanoma, keratoses, and nevi.

This curated dataset allows the web application to analyze user-uploaded images by comparing them with real clinical cases, improving prediction accuracy and providing reliable guidance.
